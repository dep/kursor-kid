# Kiki Toss Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let the user grab and fling Kiki across the screen; she follows a parabolic arc, spins mid-air, squishes on landing, and staggers dizzily before recovering.

**Architecture:** Two new `BuddyState` cases (`.tossed`, `.dizzy`) are added to `BehaviorEngine`; a velocity ring buffer in `BuddyScene` computes the launch vector from recent drag deltas; frame-by-frame physics in `BuddyScene.update(_:)` drives position while airborne; a new `stars` badge texture renders during dizzy.

**Tech Stack:** Swift, SpriteKit, XCTest — no new dependencies.

## Task 4: Velocity ring buffer and throw detection in `BuddyScene.swift`

**Files:**
- Modify: `Sources/KursorKid/BuddyScene.swift`

- [ ] **Step 1: Add ring buffer properties**

After `private var mouseDownPoint: CGPoint?`, add:

```swift
private var isTossed = false
private let tossVelocityThreshold: CGFloat = 80   // pt/s
private let gravityAccel: CGFloat = 600            // pt/s²
private var tossVelocity: CGPoint = .zero
private var dragSamples: [(point: CGPoint, time: TimeInterval)] = []
private let dragSampleWindow: TimeInterval = 0.08  // 80ms
```

- [ ] **Step 2: Accumulate drag samples in `mouseDragged`**

In `mouseDragged(with:)`, after `engine.handle(.dragStarted)` / `syncState` block, and before the end of the method, add sample accumulation. The full updated `mouseDragged` should be:

```swift
override func mouseDragged(with event: NSEvent) {
    guard let start = mouseDownPoint else { return }
    let location = event.location(in: self)
    let now = event.timestamp
    if !isDragging, hypot(location.x - start.x, location.y - start.y) > 6 {
        isDragging = true
        dragSamples.removeAll()
        sprite.removeAction(forKey: "move")
        sprite.removeAction(forKey: "anim")
        sprite.removeAction(forKey: "bounce")
        engine.handle(.dragStarted)
        syncState(now: CACurrentMediaTime())
    }
    if isDragging {
        dragSamples.append((point: location, time: now))
        // Keep only samples within the rolling window.
        dragSamples.removeAll { now - $0.time > dragSampleWindow }
    }
}
```

- [ ] **Step 3: Compute velocity and trigger toss in `mouseUp`**

Replace the existing `mouseUp` method with:

```swift
override func mouseUp(with event: NSEvent) {
    defer { mouseDownPoint = nil }
    if isDragging {
        isDragging = false
        let velocity = computeThrowVelocity(at: event.timestamp)
        let speed = hypot(velocity.x, velocity.y)
        if speed >= tossVelocityThreshold {
            isTossed = true
            tossVelocity = velocity
            engine.handle(.tossed)
            syncState(now: CACurrentMediaTime())
            spawnSpeedLines(direction: velocity)
        } else {
            engine.handle(.dragEnded(now: CACurrentMediaTime()))
            syncState(now: CACurrentMediaTime())
        }
        dragSamples.removeAll()
    } else if mouseDownPoint != nil {
        engine.handle(.clicked(now: CACurrentMediaTime()))
        syncState(now: CACurrentMediaTime())
        if bubble.isSticky {
            bubble.dismiss()
        } else {
            onBoop?()
        }
    }
}
```

Add the velocity helper after `mouseUp`:

```swift
private func computeThrowVelocity(at now: TimeInterval) -> CGPoint {
    guard dragSamples.count >= 2 else { return .zero }
    let recent = dragSamples.filter { now - $0.time <= dragSampleWindow }
    guard let first = recent.first, let last = recent.last,
          last.time > first.time else { return .zero }
    let dt = last.time - first.time
    return CGPoint(
        x: (last.point.x - first.point.x) / dt,
        y: (last.point.y - first.point.y) / dt
    )
}
```

- [ ] **Step 2: Add physics tick to `update(_:)`**

In `update(_ currentTime: TimeInterval)`, add a physics block at the very top, before the cursor/engine tick:

```swift
let dt = lastUpdateTime > 0 ? min(currentTime - lastUpdateTime, 0.05) : 0
lastUpdateTime = currentTime

if isTossed {
    tossVelocity.y -= gravityAccel * dt
    sprite.position.x += tossVelocity.x * dt
    sprite.position.y += tossVelocity.y * dt

    if sprite.position.y <= groundY {
        sprite.position.y = groundY
        isTossed = false
        engine.handle(.landed(now: currentTime))
        syncState(now: currentTime)
        land()
    }
    // Skip normal engine tick while airborne.
    return
}
```
